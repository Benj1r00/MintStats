# MintStat 🍃 - League of Legends Player Tracker

MintStat — це клієнтський веб-застосунок, створений за допомогою React, який дозволяє шукати гравців League of Legends за їхнім Riot ID та переглядати детальну статистику їхніх останніх матчів.

## 🌟 Основний функціонал

- **Пошук за Riot ID:** Зручний пошук гравців у форматі `Name#Tag` (наприклад, *Faker#KR1*).
- **Історія матчів:** Відображення 5 останніх ігор гравця.
- **Детальна статистика:** Кожна картка матчу містить:
  - Зіграного чемпіона та його рівень.
  - KDA.
  - Кількість добитих міньйонів (CS) та CS/хв.
  - Тривалість гри та результат (Перемога / Поразка).
- **Інтеграція рун та заклинань:** Відображення іконок вибраних Summoner Spells, ключової руни та другорядної гілки.

## 🚀 Технології

- **Frontend:** React
- **Стилізація:** Custom CSS
- **API:** Офіційний [Riot Games API](https://developer.riotgames.com/)

Щоб запустити цей проект на своєму комп'ютері, виконайте наступні кроки:

1. **Склонуйте репозиторій та встановіть залежності:**
   ```bash
   git clone [https://github.com/ВашЮзернейм/MintStat.git](https://github.com/ВашЮзернейм/MintStat.git)
   cd MintStat
   npm install

2. **Налаштуйте Riot API Key**
    - Перейдіть на портал розробників Riot Games та отримайте свій тимчасовий RGAPI ключ
    - У файлі src/hooks/useRiotApi.js замініть значення константи RIOT_API_KEY на свій діючий ключ.

1. **Стартуйте проект<3**
   ```bash
   npm start
